## Como abrir en Mac

Desde la raiz del proyecto:

```bash
open design/propuestas
```

O abrir cada imagen directo:

```bash
open design/propuestas/paletas_color_ui.svg
open design/propuestas/mockups_ux_ui.svg
open design/propuestas/componentes_ui.svg
open design/propuestas/clasificacion_comercios.svg
open design/propuestas/login_preview.svg
open design/propuestas/icono_app_nuevo.svg
open design/propuestas/icono_marca_nuevo.svg
open design/propuestas/icono_adaptive_foreground_nuevo.svg
open design/propuestas/splash_nuevo.svg
```

## Uso recomendado

1. Elegir una paleta principal de `paletas_color_ui.svg`.
2. Validar flujo y jerarquia visual con `mockups_ux_ui.svg`.
3. Aplicar componentes y estilos base de `componentes_ui.svg`.

Si quieres, en el siguiente paso implemento la opcion elegida directamente en `styles/theme.js` y en los componentes principales.
